***
**Использование:**

Сформирует *.csr
 ```bash */
 MyProgram -j [путь до JKS файла] [Пароль]
 ```
Сформирует *.new.p12
```bash */
MyProgram -c [путь до JKS файла] [Пароль] [путь до CRT файла ключа] [путь до CSR файла для сверки]
```

**НЮАНС:** *Пробелы в названии папок не поддерживаются внешними ресурсами*
